About

Hi there! My name is Juanita Pinkney. I am a Full Stack Web Developer, a life-long seeker of knowledge, style guru and a lover of laughter! Possibly because I choose to see the beauty in all things. For many years I have had a great interest in publication layouts and web design, as I found myself designing web pages for fun.

I recently graduated from Georgia Institute of Technology Full Stack Coding Program. Which I have acquired proficient software engineering knowledge and experience in building web and mobile applications from scratch, using Agile SCRUM methodologies with technologies such as: HTML5, CSS3, Javascript, JQuery, Bootstrap, Sass, ES6, JSX, Handlebars, AJAX, JSON, Database Management using MERN Stack - MongoDB (Mongoose), Express, React, NodeJS, and MySQL (Sequelize).

A Little About Me

Hi there! My name is Juanita Pinkney. I am a Full Stack Web Developer, a life-long seeker of knowledge, style guru, and lover of laughter! My background is in magazine editorial, commercial advertisement, and publication layout. Naturally, I progressed towards web design with UI and UX design concepts and I am now obsessed with web development.

A graduate of Georgia Institute of Technology Full Stack Flex Web Development Program, where I acquired proficient software engineering knowledge and experience in building multiple high performing, scalable, multi-client responsive web applications in an Agile development environment. Participated in all stages of software development - from early brainstorming, wireframing, prototyping and estimating, project management, well-organized coding and debugging, including tests, while following coding standards and best practices and disciplines, to deployment.

Some of the languages and technologies I use are HTML5, CSS3, Javascript, JQuery, Bootstrap, Materialize, Sass, Less, ReactStrap, ES6, JSX, Media Queries, MERN Stack - MongoDB, ExpressJS, ReactJS, NodeJS, APIs, JSON, REST, and AJAX.

Crystal Collector

Welcome to Crystal Collector! An interactive game; each value of the crystal is hidden from the user until each crystal is clicked. By clicking on a crystal, the user will add a random number of points to their total score. By matching the total score to the random number, the user will win the game. If the user's total score goes above the random number, they will lose the game. Each new game will generate a random value for each crystal.

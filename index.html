About

Front End Developer - Back End Developer - Web Designer

Hi there! My name is Juanita Pinkney. I am a Full Stack Web Developer, a life-long seeker of knowledge, style guru, and lover of laughter! With a background in magazine editorial, commercial advertisement, and publication layout, the natural progression, artistically for me was, indeed, web design. And only then, my passion for this creative outlet led to self-taught coding with HTML and CSS, playing around on MySpace. Although UI and UX design concepts fascinated me, I became obsessed with my quest for a more in-depth understanding of the process of web development.

As a recent graduate of Georgia Institute of Technology Full Stack Flex Web Development Program, I have acquired proficient software engineering knowledge and experience in building multiple high performing, scalable, multi-client responsive web applications in an Agile development environment. Participated in all stages of software development - from early brainstorming, wireframing, prototyping and estimating, project management, well-organized coding and debugging, including tests, while following coding standards and best practices and disciplines, to deployment.

Along the way, I learned mulitple languages and technologies such as; - HTML5, CSS3, Javascript, JQuery, Bootstrap, Materialize, Sass, Ant Design, Less, ES6, JSX, Handlebars - Database Management: MySQL (ORM - Sequelize), MERN Stack - MongoDB (ORM -Mongoose), Express, React, NodeJS - Media Queries, APIs, JSON, REST, AJAX - Scraping using Cheerios, Axios - Project Managment and Deployment: GIT, Github, Trello, and Heroku.

Crystal Collector

Welcome to Crystal Collector! An interactive game; each value of the crystal is hidden from the user until each crystal is clicked. By clicking on a crystal, the user will add a random number of points to their total score. By matching the total score to the random number, the user will win the game. If the user's total score goes above the random number, they will lose the game. Each new game will generate a random value for each crystal.

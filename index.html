About

Hi there! I'm Juanita, a Full Stack Front End Web Developer, life-long seeker of knowledge, minimalist style guru, and lover of laughter! My background as a versatile creative professional with over 15 years of experience in exquisite online and offline design, branding, social media, publication layout support in magazine editorial, and commercial advertisement, naturally, progressed my creative path to UI/UX web design concepts and now, obsessed with web development.

I am captivated by building great user interfaces, learning new software languages and technologies while placing great emphasis on simplistic user/client solution-based concepts. As a graduate of Georgia Institute of Technology Full Stack Flex Web Development Program, I acquired skillful software engineering knowledge in building multiple high performing, scalable, multi-client responsive web applications in an Agile SCRUM project development environment.

Disciplines, tools and technologies I currently use: HTML, CSS, Javascript, jQuery, Bootstrap, Materialize, ES6, Media Queries, WordPress, AngularJS, ReactJS, MongoDB, ExpressJS, NodeJS, Google Analytics, APIs, AJAX, JSON, CRUD, RESTful, POST, GET, Agile Scrum Methodology, Sublime, Visual Studio Code, Git, GitHub, Heroku, Trello, CodeShare, Slack, and Zoom.
